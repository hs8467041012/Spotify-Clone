This repository contains a Spotify Clone, a simple web-based music player built using HTML, CSS, and JavaScript.

 Project Overview
The Spotify Clone is a frontend-only project that demonstrates how a music streaming interface works.
It includes song playback controls, album art, and a clean UI inspired by Spotify.

This project is ideal for learning JavaScript audio handling and frontend UI development.

 Features:
 Play / Pause music
 Next & Previous track controls
 Multiple songs with album images
 Spotify-inspired UI

 Lightweight and fast
